Courier Distribution System (Kurye Dağıtım Sistemi)
Bu proje, Sakarya Üniversitesi Bilişim Sistemleri Mühendisliği bölümündeki eğitimim sürecinde, programlama temellerini ve C++ yetkinliklerini pekiştirmek amacıyla geliştirilmiştir.

Proje Hakkında
Uygulama, bir kurye dağıtım senaryosunu simüle eder. Müşterilere rastgele sipariş tutarları atar, bir dağıtım rotası oluşturur ve bu verileri dosya sistemine kaydederek analiz eder.

Teknik Özellikler ve Kazanımlar
Bu projeyi geliştirirken aşağıdaki kavramları aktif olarak kullandım:

Nesne Yönelimli Programlama (OOP): struct ve class yapıları ile veri modelleme.

Dinamik Bellek Yönetimi: new ve delete[] operatörleri ile çalışma zamanı bellek yönetimi.

Dosya İşlemleri (I/O): fstream kütüphanesi ile verilerin .txt dosyasına kalıcı olarak kaydedilmesi.

Algoritma ve Veri Analizi: * Rastgele dağıtım rotası oluşturma (shuffling).

std::min_element ve std::max_element ile en düşük/yüksek siparişlerin tespiti.

Konsol Görselleştirme: Windows API (windows.h) kullanarak terminal üzerinde renklendirme ve kullanıcı deneyimi iyileştirmesi.

Dosya Yapısı
pgödev1.cpp: Projenin tüm mantığını ve sınıflarını içeren ana kaynak kod dosyası.

siparisler.txt: Program çalıştırıldığında oluşan sipariş listesi ve rota raporu.
